------- Considerações para o funcionamento correto do bot-----

 OBS: Para multi-account de até 3 janelas só funciona no Google Chrome
 e no Windows, caso contrário, irá funcionar somente em 1 janela.

 1 - Instale as dependências do Python usando o seguinte comando no
 prompt de comando
 
  	cd "caminho da pasta do bot salvo"
 
 	pip install -r requirements.txt

 2 - O bot funciona em até 3 janelas simultanêas do Google Chrome

 3 - Sugiro trocar as imagens salva na pasta "Imagens", devido as
 diferentes resoluções de monitores, o bot não reconhece corretamente
 as imagens.

 4 - Ao iniciar o Bot, irá perguntar quantas janelas estão abertas,
digite a quantidade e o bot faz o resto.
